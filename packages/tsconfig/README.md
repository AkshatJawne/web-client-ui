# `@deephaven/tsconfig`

The tsconfig.json settings used by Deephaven Data Labs.

## Usage

Add the package to your "devDependencies":

```
npm install --save-dev @deephaven/tsconfig
```

Add to your tsconfig.json:

```
"extends": "@deephaven/tsconfig"
```
